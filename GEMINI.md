# GEMINI Agent Config

This file defines configuration and setup instructions for the **AI Agent** in this Django project.

## Project Overview
- **Backend**: Django
- **Frontend**: [Tabler](https://tabler.io/) (CSS framework)
- **Package Manager**: [uv](https://docs.astral.sh/uv/)  
  (Instead of `pip`, use `uv add` for installing dependencies)
- **Active Virtual env**: source .venv/bin/activate
